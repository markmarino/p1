My Cheat Sheet for Getting P1 Ready

    Create a new remote repository on Github.com

  • See instructions here:
    • 05_Setup_a_repository.md
  • Github URL for new repository: git@github.com:markmarino/p1.git
  • Initialize local repo:
  • In cmder:
    • $ cd E:\MAMP\htdocs\p1
    • $ git init
    • $ git remote -v
  • Connect local repo to Github repo:
  • Use this command line in cmder: $ git remote add origin git@github.com:markmarino/p1.git

    • Push up the first commit

  • See instructions here:
    • 06_Tracking_changes.md
  • First, check status. In cmder: $ git status
  • Now add the files to the repo. In cmder: $ git add --all
  • Now commit the untracked file(s)
  • In cmder: $ git commit -m "Initial project commit."
  • Finally, push the files to github. In cmder: $ git push origin master

    • Clone repo on Digital Ocean Droplet

  • See instructions here:
    • 07_Deploy_to_Digital_Ocean.md
  • If you're not logged into Digital Ocean already, fire up another cmder window and ssh into it: $ ssh root@192.241.176.184
  • Go to the Droplet root: # cd /var/www/html
  • Clone the P1 repository. In cmder: # git clone git@github.com:markmarino/p1.git

    • View the live website on Droplet

  • You can view the website at this URL: http://192.241.176.184/p1/

    • Wash, Rinse, Repeat

  • Now to make further updates, do these steps on the local server to update the files on github:
    • $ git add --all
    • $ git commit -m "Add comment here."
    • $ git push origin master
  • And then in the Droplet cmder window, pull the files:
  • # git pull origin master

    • Setup the FQDNs for the Projects

  • See instructions here:
    • 08_Live_domain_setup.md
  • First, On Namecheap, I added DNS A record for p1.markmarinorocks.me
  • I also added A records for p2,p3,p4
  • Now add the Virtual Host on my droplet so we can access the project by the FQDN p1.markmarinorocks.me
  • In order to do that, I added the following code to the /etc/apache2/sites-enabled/000-default.conf file.
    • <VirtualHost *:80>
    ServerName p1.markmarinorocks.me
    DocumentRoot "/var/www/html/p1"
    <Directory "/var/www/html/p1">
        AllowOverride All
    </Directory>
</VirtualHost>
  • I also added the virtual host entries for p2,p3,p4.
  • Now we can get to the project via a FQDN URL: http://p1.markmarinorocks.me/
