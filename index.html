Cheat Sheet for P1

  • Create a new remote repository on Github.com
  • See instructions here: https://github.com/susanBuck/dwa15-spring2016-notes/blob/master/01_Servers_and_Git/05_Setup_a_repository.md
  • Github URL for new repository: git@github.com:markmarino/p1.git
  • Initialize local repo:
  • In cmder:
    • $ cd E:\MAMP\htdocs\p1
    • $ git init
    • $ git remote -v
  • Connect local repo to Github repo:
  • Use this command line in cmder: $git remote add origin git@github.com:markmarino/p1.git
